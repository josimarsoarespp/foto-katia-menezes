- `index.html`
- `style.css`
- `script.js`
- `assets/moldura-katia-v3.png`

## Como publicar no GitHub Pages

1. Crie um repositório público no GitHub.
2. Envie todos os arquivos deste projeto para a raiz do repositório.
3. Vá em **Settings > Pages**.
4. Em **Source**, selecione **Deploy from a branch**.
5. Em **Branch**, escolha `main` e `/root`.
6. Clique em **Save**.
7. Aguarde o deploy ficar verde em **Actions**.

## Importante

A câmera só funciona em HTTPS. O GitHub Pages já gera link HTTPS automaticamente.

Use o link assim:

`https://SEU-USUARIO.github.io/NOME-DO-REPOSITORIO/`

Não teste abrindo o arquivo `index.html` direto do computador, porque a câmera não abre em `file://`.
